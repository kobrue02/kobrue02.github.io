Stock Market Prediction Tool

This project is currently in progress.
We are attempting to train a model to predict prices in the stock market based on the sentiment analysis of social media.
There is a bit of research on this topic already, like this paper by Pagolu et al. and this paper by Khedr et al., in which the authors were able to train those prediction models with an accuracy of up to 87%.
Currently, the tool has an interface for reddit, twitter, stocktwits, news headlines and 4chan and performs sentiment analysis on the data scrapped from those.
However, we are still working on the model.

Gentify

Gentify is a spotify playlist generation tool built in Python using the Spotipy library.
The user gets a list of genres to choose from (up to 5) and the script generates a playlist based off those genres.
